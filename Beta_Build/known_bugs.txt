StepSmart Known Bugs & Limitations

1. Application assumes valid BLE data is received; partial, delayed, or inaccurate data is not handled.
2. Pressure data is limited to three sensor points.
3. Pressure averages are calculated after disconnecting from device, preventing the user from viewing real-time updates.
4. Mobile application is supported on Android devices only.
5. Automatic reconnection is not supported if BLE connection is lost or interrupted.
6. Step count is not always accurate.
7. Application is not a standalone mobile app; the development server must be started from a terminal before connecting a mobile device.
